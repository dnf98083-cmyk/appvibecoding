# PRD: 개인 할 일 관리 앱

## 제품 개요
매일 10~20개의 할 일을 관리하는 개인용 웹 앱.
브라우저에서 바로 실행되며, LocalStorage를 통해 데이터가 영구 저장된다.

- **기술 스택**: 순수 HTML + CSS + JavaScript (라이브러리 없음)
- **저장 방식**: LocalStorage (새로고침, 재시작 후에도 유지)
- **실행 방법**: index.html 파일을 브라우저에서 열기

| 환경 | 데스크톱 브라우저 (Chrome, Edge 등) |

## 핵심 기능 요구사항

### 1. 할 일 추가
- 텍스트 입력창에 내용 작성 후 Enter 또는 [추가] 버튼으로 등록
- 등록 시 카테고리(업무/개인/공부) 선택
- 마감일(선택 사항) 설정 가능

### 2. 할 일 수정
- 등록된 항목 클릭 시 인라인 편집 가능
- 내용, 카테고리, 마감일 수정 가능
- 수정 완료는 Enter 또는 저장 버튼

### 3. 할 일 삭제
- 각 항목에 삭제(✕) 버튼 배치
- 완료된 항목 전체 삭제 버튼 제공

### 4. 완료 체크
- 체크박스 클릭으로 완료/미완료 토글
- 완료 시 텍스트에 취소선 표시
- 완료 항목은 목록 하단으로 자동 이동

### 5. 카테고리 분류
| 카테고리 | 색상 |
|----------|------|
| 업무 | 파란색 |
| 개인 | 초록색 |
| 공부 | 보라색 |

- 상단 카테고리 탭으로 필터링 (전체 / 업무 / 개인 / 공부)

### 6. 진행률 보기
- 상단에 전체 진행률 바 표시 (완료 수 / 전체 수)
- 카테고리별 완료율 숫자 표시
- 오늘 날짜 기준 통계 제공

## 단계별 구현 계획

### 단계 1 — 핵심 CRUD
- 할 일 추가 / 삭제
- 완료 체크
- LocalStorage 저장
- **완료 기준**: 항목을 추가하고 새로고침해도 남아있는가

### 단계 2 — 카테고리 & 필터
- 카테고리 선택 및 뱃지 표시
- 탭 필터링 (전체/업무/개인/공부)
- 완료 항목 하단 이동
- **완료 기준**: 카테고리별로 필터링이 작동하는가

### 단계 3 — 진행률 & UI 완성
- 전체 진행률 바
- 카테고리별 통계
- 인라인 수정
- 완료 항목 일괄 삭제
- 마감일 설정 및 D-day 표시
- **완료 기준**: 진행률이 실시간으로 업데이트되는가
