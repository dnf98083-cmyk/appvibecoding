# /quiz-add — 새 퀴즈 문항 추가

새 문항을 생성하고 할루시네이션 검증 후 questions.json에 추가합니다.

## 사용법

```
/quiz-add [카테고리] [문제수]
```

예시:
- `/quiz-add 한국사 3` → 한국사 문항 3개 추가
- `/quiz-add 과학` → 과학 문항 1개 추가

## 실행 단계

1. **현재 문제 은행 확인** — questions.json의 카테고리별 문항 수 파악

2. **문항 생성** — 아래 형식 준수:
```json
{
  "id": [다음 번호],
  "category": "[카테고리]",
  "question": "[문제]",
  "choices": ["정답", "오답1", "오답2", "오답3"],
  "answer": 0,
  "explanation": "[해설]"
}
```

3. **자체 검증 (CLAUDE.md 체크리스트 적용)**
   - [ ] 정답이 명확히 하나인가?
   - [ ] 선택지가 정확히 4개인가?
   - [ ] 해설이 충분한가?

4. **questions.json에 추가** — questions 배열 끝에 삽입

5. **검증 실행**
```
python -X utf8 validator.py
```

## 주의사항
- 카테고리는 반드시 **한국사 / 과학 / 지리 / 일반상식** 중 하나
- AI가 생성한 수치/통계는 반드시 공식 자료와 대조 후 추가
