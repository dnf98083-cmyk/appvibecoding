# /quiz-range — 범위 지정 퀴즈 실행

카테고리 또는 문제 수 범위를 지정해서 퀴즈를 실행합니다.

## 사용법

```
/quiz-range [카테고리] [문제수]
```

예시:
- `/quiz-range 한국사 5` → 한국사 5문제
- `/quiz-range 과학` → 과학 전체(10문제)
- `/quiz-range 전체 10` → 전체 카테고리 랜덤 10문제

## 실행 단계

1. questions.json에서 해당 카테고리 문제 필터링
2. 요청한 수만큼 랜덤 선택 (없으면 전체)
3. 아래 명령으로 실행:

```
python -X utf8 quiz_game.py
```

카테고리/문제수 인자를 반영해서 게임을 바로 시작합니다.
카테고리 목록: **한국사 / 과학 / 지리 / 일반상식 / 전체**
