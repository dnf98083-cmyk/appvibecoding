# /quiz-report — 게임 결과 분석 리포트

results/ 폴더의 게임 기록을 분석하고 학습 리포트를 생성합니다.

## 실행 단계

1. **결과 파일 수집**
   results/ 폴더의 모든 JSON 파일을 읽습니다.
   파일이 없으면 "기록 없음"을 안내합니다.

2. **통계 계산**
   - 전체 게임 횟수
   - 평균 점수 및 최고/최저 점수
   - 카테고리별 정답률 (약점 카테고리 파악)
   - 난이도별 정답률
   - 평균 응답 시간

3. **리포트 출력 형식**
   ```
   ===== 학습 분석 리포트 =====
   총 게임: X회 | 평균 점수: XX%
   
   카테고리별 정답률:
     역사: XX% [████████░░]
     과학: XX% [██████████]
     지리: XX% [██████░░░░]
     문화: XX% [███████░░░]
   
   취약 영역: [카테고리명]
   권장 학습: [구체적인 학습 제안]
   ```

4. **맞춤 학습 제안**
   - 정답률 50% 미만 카테고리에 집중 학습 권장
   - 낮은 신뢰도 문항에서 틀린 경우 해당 주제 재검토 권장
   - 응답 시간이 긴 문항 유형 파악

결과 파일이 없으면 게임을 먼저 실행하도록 안내합니다: `python quiz_game.py`
