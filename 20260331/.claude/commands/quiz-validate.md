# /quiz-validate — 문제 은행 유효성 검사

questions.json을 읽고 아래 항목을 전부 검사해줘.

1. 총 문제 수가 40개인가?
2. 카테고리가 정확히 한국사/과학/지리/일반상식 4개인가?
3. 각 카테고리마다 정확히 10문제씩인가?
4. 모든 문제에 choices가 4개씩 있는가?
5. 모든 문제의 answer가 0~3 범위인가?
6. 모든 문제에 explanation이 있는가?
7. 문제 id가 중복 없이 1~40인가?

그리고 pytest tests/test_quiz.py 를 실행해서 결과도 함께 보여줘.

검사 결과를 표로 정리하고, 문제가 있으면 어떤 항목(id)이 잘못됐는지 알려줘.
모두 통과하면 "✅ 문제 은행 검증 통과" 메시지를 출력해줘.
