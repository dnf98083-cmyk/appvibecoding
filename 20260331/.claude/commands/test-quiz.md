# /test-quiz — 퀴즈 게임 전체 테스트 실행

퀴즈 게임의 모든 테스트를 실행하고 결과를 요약합니다.

## 실행 단계

1. `tests/test_quiz.py` 를 pytest로 실행합니다:
   ```
   python -m pytest tests/test_quiz.py -v
   ```

2. 테스트 결과를 분석합니다:
   - 전체 테스트 수, 통과/실패 수 요약
   - 실패한 테스트가 있으면 원인을 설명합니다
   - 커버리지가 낮은 영역을 파악합니다

3. 결과를 다음 형식으로 보고합니다:
   ```
   테스트 결과: X/Y 통과
   - 통과: [테스트 목록]
   - 실패: [실패 원인 설명]
   - 권장 조치: [필요시]
   ```

실패한 테스트가 있으면 원인을 분석하고 수정 방법을 제안합니다.
