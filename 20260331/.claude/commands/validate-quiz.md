# /validate-quiz — 할루시네이션 교차 검증 실행

questions.json의 모든 문항을 검증하고 할루시네이션 리스크를 리포트합니다.

## 실행 단계

1. validator.py를 실행합니다:
   ```
   python -X utf8 validator.py
   ```

2. 리포트를 분석합니다:
   - 구조적 오류(필수 필드 누락, 인덱스 오류 등) 확인
   - 낮은 신뢰도(< 0.7) 문항 목록 추출
   - 출처가 부실한 문항 파악

3. 검증 결과를 다음 형식으로 요약합니다:
   ```
   검증 결과: X/Y 통과
   ⚠️  교차 검증 필요: [문항 목록 및 이유]
   ❌ 즉시 수정 필요: [문항 목록 및 수정 방법]
   ```

4. 낮은 신뢰도 문항이 있으면 교차 검증 방법을 구체적으로 안내합니다:
   - 어떤 자료를 찾아야 하는지
   - 신뢰도를 어떻게 높일 수 있는지
   - questions.json 수정 방법

오류가 있으면 수정 후 재검증을 권장합니다.
