# 상식 퀴즈 게임 PRD (Product Requirements Document)

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 상식 퀴즈 게임 |
| 목적 | 카테고리별 상식 퀴즈를 통해 학습하고 점수를 기록하는 CLI 게임 |
| 플랫폼 | Python CLI (터미널 실행) |

---

## 2. 게임 규칙

### 2-1. 문제 형식
- **4지선다 객관식** (보기 4개 중 1개 선택)
- 번호(1~4) 입력으로 답변

### 2-2. 카테고리 및 문제 수

| 카테고리 | 문제 수 | 예시 주제 |
|----------|---------|-----------|
| 한국사 | 10문제 | 조선, 근현대사, 인물, 사건 |
| 과학 | 10문제 | 물리, 화학, 생물, 지구과학 |
| 지리 | 10문제 | 세계지리, 한국지리, 수도, 지형 |
| 일반상식 | 10문제 | 문화, 스포츠, 시사, 언어 |
| **합계** | **40문제** | |

### 2-3. 진행 방식
- 게임 시작 시 카테고리 선택 또는 전체(랜덤) 선택
- 문제 출제 순서는 랜덤 셔플
- 각 문제 제한시간 없음 (자유 응답)

### 2-4. 피드백
- 답변 즉시 정답/오답 표시
- 오답 시 정답 및 해설 표시
- 정답 시 격려 메시지 표시

### 2-5. 점수 및 순위
- 정답 1개당 10점
- 최대 점수: 400점 (전체 40문제 기준)
- 게임 종료 후 최종 점수 + 등급 표시
- 결과를 JSON 파일로 저장하여 순위 기록 유지
- 상위 10명 랭킹 조회 가능 (닉네임 입력)

---

## 3. 기능 요구사항

### 필수 기능 (Must Have)
- [ ] 카테고리별 10문제씩 총 40문제 문제 은행
- [ ] 4지선다 문제 출제 및 입력 처리
- [ ] 정답/오답 즉시 피드백 (해설 포함)
- [ ] 최종 점수 계산 및 등급 표시
- [ ] 닉네임 입력 후 결과 저장
- [ ] 상위 10위 랭킹 보드

### 선택 기능 (Nice to Have)
- [ ] 카테고리 선택 모드 vs 전체 랜덤 모드
- [ ] 오답 노트 (틀린 문제만 재도전)
- [ ] 난이도별 표시 (★ 별점)

## 4. 등급 기준

| 점수 범위 | 등급 | 메시지 |
|-----------|------|--------|
| 380~400점 | S | 완벽해요! 상식왕! |
| 300~370점 | A | 훌륭해요! |
| 200~290점 | B | 괜찮아요! |
| 100~190점 | C | 조금 더 공부해요! |
| 0~90점 | D | 다시 도전해보세요! |

## 7. 성공 기준

- 카테고리별 10문제 × 4 = 40문제 구성 완료
- 모든 문제에 해설 포함
- 정답/오답 즉시 피드백 동작
- 랭킹 저장 및 조회 기능 동작
- pytest 테스트 통과
