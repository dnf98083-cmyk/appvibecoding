# PRD: 냉장고 재료 인식 & 레시피 추천 웹 앱

## 제품 개요
냉장고 사진을 업로드하면 AI가 재료를 자동으로 인식하고, 해당 재료로 만들 수 있는 레시피를 추천하는 웹 애플리케이션.

- **이미지 인식 모델**: `google/gemma-3-27b-it:free` (재료 추출)
- **텍스트 생성 모델**: `qwen/qwen3.6-plus:free` (레시피 추천)
- **API**: OpenRouter

---

## 단계 1: 이미지 업로드 & 재료 인식 MVP

### 목표
냉장고 사진을 받아 AI가 재료 목록을 텍스트로 반환하는 핵심 기능 구현.

### 기능 요구사항
| 기능 | 설명 |
|------|------|
| 이미지 업로드 | 파일 선택 버튼으로 로컬 이미지(.jpg, .png) 업로드 |
| 이미지 미리보기 | 업로드한 사진을 화면에 표시 |
| 재료 인식 | Gemma 모델에 이미지 전송 → 재료 목록 추출 |
| 결과 표시 | 인식된 재료를 리스트 형태로 화면에 출력 |

### 화면 구성
```
[냉장고 사진 업로드 버튼]
[이미지 미리보기 영역]
[재료 인식하기 버튼]
[인식된 재료 목록]
  - 당근
  - 달걀
  - 우유 ...
```

### 기술 스펙
- **파일**: `index.html` 단일 파일 (HTML + CSS + JS)
- **이미지 → base64** 변환 후 OpenRouter API 전송
- **프롬프트**: "이 냉장고 사진에서 보이는 식재료를 한국어로 목록만 나열해줘."

### 완료 기준
- [ ] 이미지 업로드 및 미리보기 작동
- [ ] Gemma 모델이 재료 목록 반환
- [ ] 결과가 화면에 정상 출력

## 단계 2: 레시피 추천 기능 추가

### 목표
인식된 재료를 바탕으로 Qwen 모델이 레시피를 추천하고, 결과를 보기 좋게 표시.

### 기능 요구사항
| 기능 | 설명 |
|------|------|
| 재료 편집 | 인식된 재료 목록을 사용자가 직접 추가/삭제 가능 |
| 레시피 추천 | Qwen 모델에 재료 전송 → 레시피 3개 추천 |
| 레시피 상세 표시 | 요리명, 필요 재료, 조리 순서를 구조화하여 출력 |
| 로딩 상태 | API 호출 중 스피너 또는 로딩 메시지 표시 |

### 화면 구성
```
[재료 목록] (편집 가능)
  ✕ 당근   ✕ 달걀   ✕ 우유   [+ 직접 추가]

[레시피 추천받기 버튼]

[추천 레시피 카드 x 3]
  ┌──────────────────┐
  │ 🍳 달걀볶음밥    │
  │ 재료: 달걀, 밥   │
  │ 1. 팬에 기름 두기│
  │ 2. 달걀 풀어 넣기│
  └──────────────────┘
```

### 기술 스펙
- **프롬프트**: "재료: {재료목록}. 이 재료들로 만들 수 있는 한식 레시피 3가지를 요리명, 재료, 조리순서 형식으로 알려줘."
- 단계 1의 `index.html`에 기능 추가 (파일 분리 없음)

### 완료 기준
- [ ] 재료 추가/삭제 가능
- [ ] Qwen 모델이 레시피 3개 반환
- [ ] 레시피 카드 형태로 화면에 출력
- [ ] 로딩 중 UI 표시

## 단계 3: UI/UX 완성 & 사용성 개선

### 목표
실제 서비스처럼 사용할 수 있는 완성도 높은 UI 구현.

### 기능 요구사항
| 기능 | 설명 |
|------|------|
| 드래그 앤 드롭 | 이미지를 끌어다 놓아 업로드 |
| 에러 처리 | API 실패 시 사용자 친화적 오류 메시지 표시 |
| 레시피 즐겨찾기 | 마음에 드는 레시피를 LocalStorage에 저장 |
| 반응형 디자인 | 모바일 화면에서도 정상 동작 |
| 재분석 | 같은 재료로 레시피 재추천 버튼 |

### 화면 구성
```
┌─────────────────────────────────┐
│  🧊 냉장고 레시피 추천          │
├─────────────────────────────────┤
│  [📷 사진을 여기에 끌어다 놓기] │
│       또는 클릭하여 선택        │
├─────────────────────────────────┤
│  인식된 재료  [재분석 🔄]       │
│  당근 ✕  달걀 ✕  우유 ✕        │
│  [+ 재료 추가]                  │
├─────────────────────────────────┤
│  추천 레시피          [즐겨찾기]│
│  카드 1 / 카드 2 / 카드 3       │
└─────────────────────────────────┘
```

### 기술 스펙
- **스타일**: CSS만 사용 (외부 라이브러리 없음)
- **즐겨찾기**: `localStorage`에 JSON 저장
- **에러 메시지**: rate limit → "잠시 후 다시 시도해주세요", 네트워크 오류 → "인터넷 연결을 확인해주세요"

### 완료 기준
- [ ] 드래그 앤 드롭 업로드 작동
- [ ] 에러 상황별 메시지 출력
- [ ] 즐겨찾기 저장/불러오기 작동
- [ ] 모바일(375px) 화면에서 레이아웃 정상
