# PRD 단계 1: 이미지 업로드 & 재료 인식 MVP

## 제품 개요
냉장고 사진을 업로드하면 AI가 재료를 자동으로 인식하는 핵심 기능 구현.

- **이미지 인식 모델**: `google/gemma-3-27b-it:free`
- **API**: OpenRouter

---

## 목표
냉장고 사진을 받아 AI가 재료 목록을 텍스트로 반환하는 MVP 완성.

---

## 기능 요구사항

| 기능 | 설명 |
|------|------|
| 이미지 업로드 | 파일 선택 버튼으로 로컬 이미지(.jpg, .png) 업로드 |
| 이미지 미리보기 | 업로드한 사진을 화면에 표시 |
| 재료 인식 | Gemma 모델에 이미지 전송 → 재료 목록 추출 |
| 결과 표시 | 인식된 재료를 리스트 형태로 화면에 출력 |

---

## 화면 구성

```
[냉장고 사진 업로드 버튼]
[이미지 미리보기 영역]
[재료 인식하기 버튼]
[인식된 재료 목록]
  - 당근
  - 달걀
  - 우유 ...
```

---

## 기술 스펙

- **파일**: `index.html` 단일 파일 (HTML + CSS + JS)
- **이미지 → base64** 변환 후 OpenRouter API 전송
- **프롬프트**: "이 냉장고 사진에서 보이는 식재료를 한국어로 목록만 나열해줘."
