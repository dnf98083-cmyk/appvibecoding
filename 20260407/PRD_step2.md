# PRD 단계 2: 레시피 추천 기능 추가

## 전제 조건
단계 1(이미지 업로드 & 재료 인식)이 완료된 상태.

- **텍스트 생성 모델**: `qwen/qwen3.6-plus:free`
- **API**: OpenRouter

---

## 목표
인식된 재료를 바탕으로 Qwen 모델이 레시피를 추천하고, 결과를 보기 좋게 표시.

---

## 기능 요구사항

| 기능 | 설명 |
|------|------|
| 재료 편집 | 인식된 재료 목록을 사용자가 직접 추가/삭제 가능 |
| 레시피 추천 | Qwen 모델에 재료 전송 → 레시피 3개 추천 |
| 레시피 상세 표시 | 요리명, 필요 재료, 조리 순서를 구조화하여 출력 |
| 로딩 상태 | API 호출 중 스피너 또는 로딩 메시지 표시 |

---

## 화면 구성

```
[재료 목록] (편집 가능)
  ✕ 당근   ✕ 달걀   ✕ 우유   [+ 직접 추가]

[레시피 추천받기 버튼]

[추천 레시피 카드 x 3]
  ┌──────────────────┐
  │ 🍳 달걀볶음밥    │
  │ 재료: 달걀, 밥   │
  │ 1. 팬에 기름 두기│
  │ 2. 달걀 풀어 넣기│
  └──────────────────┘
```

- **프롬프트**: "재료: {재료목록}. 이 재료들로 만들 수 있는 한식 레시피 3가지를 요리명, 재료, 조리순서 형식으로 알려줘."
- 단계 1의 `index.html`에 기능 추가 (파일 분리 없음)
