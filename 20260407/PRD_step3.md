# PRD 단계 3: UI/UX 완성 & 사용성 개선

## 전제 조건
단계 2(레시피 추천)가 완료된 상태.

---

## 목표
실제 서비스처럼 사용할 수 있는 완성도 높은 UI 구현.

---

## 기능 요구사항

| 기능 | 설명 |
|------|------|
| 드래그 앤 드롭 | 이미지를 끌어다 놓아 업로드 |
| 에러 처리 | API 실패 시 사용자 친화적 오류 메시지 표시 |
| 레시피 즐겨찾기 | 마음에 드는 레시피를 LocalStorage에 저장 |
| 반응형 디자인 | 모바일 화면에서도 정상 동작 |
| 재추천 | 같은 재료로 레시피 재추천 버튼 |

---

## 화면 구성

```
┌─────────────────────────────────┐
│  🧊 냉장고 레시피 추천          │
├─────────────────────────────────┤
│  [📷 사진을 여기에 끌어다 놓기] │
│       또는 클릭하여 선택        │
├─────────────────────────────────┤
│  인식된 재료  [재추천 🔄]       │
│  당근 ✕  달걀 ✕  우유 ✕        │
│  [+ 재료 추가]                  │
├─────────────────────────────────┤
│  추천 레시피          [즐겨찾기]│
│  카드 1 / 카드 2 / 카드 3       │
└─────────────────────────────────┘
```

---

## 기술 스펙

- **스타일**: CSS만 사용 (외부 라이브러리 없음)
- **즐겨찾기**: `localStorage`에 JSON 저장
- **에러 메시지**:
  - rate limit → "잠시 후 다시 시도해주세요"
  - 네트워크 오류 → "인터넷 연결을 확인해주세요"
  - 인증 오류 → "API 키를 확인해주세요"
